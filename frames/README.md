# Frames Directory

This directory contains all images/frames for the assignment report.

## Required Frames:

1. **trans_matrices.png** - Output of transformation matrices (RotX, RotY, RotZ, Trans3D)
2. **plot_frame_3d.png** - 3D coordinate frames visualization
3. **plot_robot_3d.png** - 3-link robot visualization
4. **animate_frame_050.png** - Sample frame from robot animation
5. **plot_robot_flexible.png** - 4-link flexible robot
6. **arbitrary_rotation.png** - Console output of rotation sequences
7. **test_output.png** - Complete test script execution

## How to Generate:

### Option 1: Run automated capture script
```matlab
cd('D:/Masters/Robotics/assignment2')
capture_frames
```

### Option 2: Follow SCREENSHOT_GUIDE.md
See the parent directory's SCREENSHOT_GUIDE.md for manual capture instructions.

## Notes:

- All PNG files should be at least 150 DPI for good quality
- Use consistent viewing angle (45°, 30°) for 3D plots
- For console outputs, capture the terminal window or convert text files to images
